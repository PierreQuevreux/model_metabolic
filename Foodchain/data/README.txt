Files containing the data

###########
chronic.txt : File containing the dynamics
t : time (year)
K : carrying capacity
X : metabolic adjustment coefficient
PP : primary producers production
SP : secondary production
Xi : biomass of species i


###########
bifurcation.txt : File containing the bifurcation from N to TR
K : carrying capacity
X : metabolic adjustment coefficient
value : value of the local extremum
variable : measured variable (N, D, P...)
